◆ abacus_init()

int abacus_init ( uint8_t abacus_version,
uint8_t abacus_clockSpeed )

Initiates all the variables and hardware of the board and on the clock speed decided by the user:

  • Hardware versions:
    • AB_ABACUSVERSION2013,
  • AB_ABACUSVERSION2014,
  • AB_ABACUSVERSION2017

Clock speeds only applies to MCLK:

  • AB_CLOCK1MHZ using internal RC
  • AB_CLOCK8MHZ using internal RC
  • AB_CLOCK20MHZ using external xTal
  • AB_CLOCK25MHZ using external xtal Keep in mind that ACLK is always set to 32.768kHz. SMCLK = 20MHz when MSCLK selected as 20MHz or 25MHz, otherwise SMCLK = MCLK

◆ abacus_init_clock()

void abacus_init_clock ( uint8_t abacus_clockSpeed)

You can select the internal clock to 8MHz or 1MHz or external clock to 20MHz or 25MHz This will affect timers and serial ports speeds but is already taken care of in the libraries.
